# Linked List

- ### Node structure :

```
struct node {
    int data;
    struct node* next;
}
```

## Types :

- ### Singly Linked List :

```
struct node {
    int data;
    struct node* next;
}
```

```
struct node {
    int data;
    struct node* prev;

}
```
